## Qué hace
- Crea y modifica archivos Excel (.xlsx) con formato profesional
- Escribe fórmulas, funciones y validación de datos
- Genera gráficos, tablas dinámicas y formato condicional
- Transforma y limpia datos dentro de hojas de cálculo
- Importa/exporta datos entre CSV, JSON y Excel

## Cuándo usarlo
- Cuando necesites generar reportes en Excel automáticamente
- Para transformar o limpiar datos tabulares
- Al crear dashboards en Excel con gráficos y tablas dinámicas
- Para automatizar procesos manuales de hojas de cálculo

## Cómo usarlo
1. Describe los datos, el análisis o el reporte deseado
2. Proporciona los datos de entrada (o descripción de estos)
3. Se generará el archivo .xlsx con fórmulas y formato incluidos

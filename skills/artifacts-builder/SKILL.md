## Qué hace
- Genera componentes React autónomos con Tailwind CSS y shadcn/ui
- Crea artefactos HTML completos embeddables (dashboards, charts, formularios)
- Implementa diseños responsivos y accesibles por defecto
- Incluye interactividad con estado local y efectos
- Produce código limpio y listo para integrar en proyectos existentes

## Cuándo usarlo
- Cuando necesites prototipar una UI rápidamente
- Para crear dashboards, paneles de administración o reportes visuales
- Cuando quieras generar componentes UI descartables pero funcionales
- Para explorar visualmente ideas de diseño antes de implementarlas

## Cómo usarlo
1. Describe el artefacto que necesitas (tipo, funcionalidad, datos)
2. Especifica el stack (React con o sin TS, Tailwind, shadcn/ui)
3. Se generará un HTML autónomo con todo incluido

## Qué hace
- Configura y ejecuta tests de Playwright contra aplicaciones web locales
- Navega e interactúa con la UI como lo haría un usuario real
- Verifica elementos, estados, rutas y flujos completos
- Toma screenshots y videos para depuración visual
- Genera reportes de resultados de las pruebas

## Cuándo usarlo
- Cuando necesites verificar visualmente que un cambio funciona
- Para testear flujos de usuario completos (login, registro, checkout)
- Antes de desplegar para asegurar que la UI funciona correctamente
- Cuando los tests unitarios no cubren la interacción del usuario

## Cómo usarlo
1. Asegúrate de que la aplicación local esté corriendo
2. Describe el flujo o funcionalidad a probar
3. Se generarán y ejecutarán los tests automáticamente

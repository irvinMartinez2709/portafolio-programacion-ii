## Qué hace
- Construye interfaces con clases utilitarias de Tailwind CSS
- Implementa diseño responsivo con breakpoints (sm, md, lg, xl)
- Crea temas personalizados y configura el archivo tailwind.config
- Optimiza el CSS final eliminando clases no utilizadas (purge)
- Compone patrones de UI comunes con @apply y componentes

## Cuándo usarlo
- Al diseñar cualquier interfaz con Tailwind CSS
- Cuando necesites implementar diseño responsivo consistente
- Para crear un design system o tema personalizado
- Para optimizar el bundle de CSS en producción

## Cómo usarlo
1. Describe el diseño o componente que necesitas
2. Especifica la paleta de colores o tema si aplica
3. Se generará el código con las clases de Tailwind apropiadas

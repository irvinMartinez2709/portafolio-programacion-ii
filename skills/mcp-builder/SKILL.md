## Qué hace
- Scaffolding de servidores MCP desde cero con Python o TypeScript
- Implementa tools, resources y prompts según el protocolo MCP
- Genera esquemas de validación y tipos para las herramientas
- Incluye manejo de errores, logging y transporte adecuado
- Documenta el servidor y sus herramientas para descubrimiento
- Sigue las mejores prácticas de seguridad y rendimiento

## Cuándo usarlo
- Cuando necesites exponer herramientas o datos a través de MCP
- Para integrar sistemas existentes como servidores MCP
- Al crear skills o herramientas que requieran un servidor MCP
- Para contribuir al ecosistema MCP con servidores de calidad

## Cómo usarlo
1. Describe qué funcionalidad debe exponer el servidor MCP
2. Elige Python o TypeScript como lenguaje de implementación
3. Se generará el servidor con todas las herramientas configuradas

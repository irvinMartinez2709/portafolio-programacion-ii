## Qué hace
- Audita y mejora el SEO on-page (meta tags, headings, estructura, keywords)
- Optimiza rendimiento web (Core Web Vitals, velocidad, mobile-first)
- Analiza y mejora la arquitectura de información y linking interno
- Genera estrategias de contenido optimizadas para búsqueda
- Implementa datos estructurados (schema.org) para rich snippets

## Cuándo usarlo
- Al desarrollar o rediseñar un sitio web
- Para mejorar el tráfico orgánico y posicionamiento
- Cuando necesites auditar el SEO de un sitio existente
- Al crear contenido nuevo para asegurar su visibilidad

## Cómo usarlo
1. Proporciona la URL del sitio o el contenido a optimizar
2. Describe los objetivos de SEO (palabras clave, audiencia, mercado)
3. Se generará un plan de acción con mejoras priorizadas

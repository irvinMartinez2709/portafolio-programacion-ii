## Qué hace
- Crea presentaciones PowerPoint (.pptx) desde cero
- Aplica diseños, temas, transiciones y animaciones
- Genera diapositivas con tablas, gráficos, imágenes y diagramas
- Modifica presentaciones existentes (contenido, formato, diseño)
- Extrae contenido y estructura de presentaciones

## Cuándo usarlo
- Para generar reportes visuales o presentaciones automáticas
- Al actualizar presentaciones existentes con nuevos datos
- Cuando necesites estandarizar el formato de presentaciones
- Para extraer diapositivas o contenido de presentaciones

## Cómo usarlo
1. Describe la presentación: tema, número de diapositivas, contenido
2. Especifica el diseño o plantilla a utilizar
3. Se generará el archivo .pptx listo para usar

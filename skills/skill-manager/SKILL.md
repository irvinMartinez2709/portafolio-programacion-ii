## Qué hace
- Lista todos los skills disponibles con sus descripciones y categorías
- Guía la creación interactiva de nuevos skills paso a paso
- Revisa skills existentes y sugiere mejoras de contenido o formato
- Valida que los skills cumplan con el formato oficial de OpenCode
- Ayuda a instalar skills en proyectos específicos

## Cuándo usarlo
- Para administrar la colección de skills
- Cuando necesites crear un nuevo skill y quieras guía
- Para revisar y mejorar skills existentes
- Al auditar skills para asegurar consistencia y calidad

## Cómo usarlo
1. Indica qué operación deseas: listar, crear, revisar o instalar
2. Para creación: describe la funcionalidad del nuevo skill
3. Para revisión: proporciona el skill a revisar
